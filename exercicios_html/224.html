Tríceps Peito com Barra

Exercício para fortalecimento e hipertrofia dos músculos tríceps, com enfoque o tríceps braquial.

Ficha Técnica

Tipo: Musculação

Grupo Muscular: Tríceps

Aparelho: Nenhum

Músculos: Nenhum

Como realizar

  1. Deite sobre um banco com os joelhos flexionados e pés fixos no suporte;
  2. Com as costas levemente arqueadas, estabilize os glúteos no banco;
  3. Pegue a barra com as mãos próximas uma da outra em distância similar a largura dos ombros;
  4. Posicione a barra com os braços estendidos na linha do peitoral, essa é a posição inicial do exercício;
  5. Manter os punhos retos, flexione os cotovelos para baixo, descendo a barra de maneira controlada até a altura do peitoral;
  6. Empurre a barra para cima concentrando a força nos músculos dos tríceps;
  7. Repita os movimentos, conforme o número de repetições orientado pelo o professor.
