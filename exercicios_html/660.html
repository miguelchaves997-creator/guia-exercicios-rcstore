Agachamento Isométrico

O exercício é realizado de forma isométrico com o objetivo trabalhar os músculos do glúteo, lombar, quadríceps e o abdômen.

Ficha Técnica

Tipo: Funcional

Grupo Muscular: Perna

Aparelho: Nenhum

Músculos: Nenhum

Como realizar

  1. Primeiro, estenda seus braços para frente;
  2. Posicione os seus pés na mesma distância da linha dos seus ombros;
  3. Agache para trás, para isso, empurre os quadris para trás, abra os joelhos levemente para os lados;
  4. Desça até ficar em uma posição perpendicular ou atingir um ângulo reto;
  5. Mantenha-se nessa posição o maior tempo que aguentar;
  6. Volte para a posição inicial.
