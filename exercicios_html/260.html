Prancha de Braços Jacks

Exercício para fortalecimento e resistência muscular, bem como, a resistência cardiovascular. Acelera o metabolismo e queima de gorduras.

Ficha Técnica

Tipo: Funcional

Grupo Muscular: Corpo

Aparelho: Nenhum

Músculos: Nenhum

Como realizar

  1. Deite no solo;
  2. Barriga para baixo com as pontas dos pés e as mãos apoiadas no chão;
  3. Eleve o tronco até formar uma linha reta desde os ombros até aos tornozelos, essa é posição inicial do exercício;
  4. Para executar os movimentos, dê um impulso movendo os pés para os lados e retorne rapidamente à posição inicial;
  5. Repita os movimentos, conforme o número se repetições orientado pelo professor.
