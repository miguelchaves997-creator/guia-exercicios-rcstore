Remada Unilateral no TRX

O exercício trabalha o fortalecimento dos músculos das costas, os ombros e bíceps, como também a estabilização. Esse exercício é recomendado para praticantes avançados.

Ficha Técnica

Tipo: Funcional

Grupo Muscular: Costas

Aparelho: Nenhum

Músculos: Nenhum

Como realizar

  1. Agarre com uma mão a fita com a palma virada para dentro e incline seu corpo para trás, trazendo o peso para os calcanhares;
  2. Permaneça com o braço estendido para frente, no formato de uma diagonal;
  3. Dobre o cotovelo e puxe, alinhando a mão ao tórax, em movimento de uma remada;
  4. Retorne à posição inicial e repita os movimentos.
