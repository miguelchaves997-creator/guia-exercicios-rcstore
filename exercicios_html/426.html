Remada Aberta no TRX

É um excelente exercício de musculação para treinar os músculos das costas como o grande dorsal, trapézio, redondos e romboides.

Ficha Técnica

Tipo: Funcional

Grupo Muscular: Costas

Aparelho: Nenhum

Músculos: Nenhum

Como realizar

  1. Conecte o TRX firmemente a uma barra;
  2. Segure as alças de forma que as palmas das mãos estejam voltadas para baixo;
  3. Caminhe para frente para se inclinar e os braços estejam completamente estendidos à sua frente;
  4. Mantenha o tronco firme e encaixado e puxe os cotovelos para trás enquanto eleva o corpo até as alças;
  5. Faça uma pausa quando o tórax estiver a poucos centímetros das alças;
  6. Abaixe o corpo de volta à posição inicial.
