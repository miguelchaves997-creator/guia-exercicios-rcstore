Lateral Alongada

Exercício de alongamento para as dorsais, preparando-as para a atividade física.

Ficha Técnica

Tipo: Alongamento

Grupo Muscular: Costas

Aparelho: Nenhum

Músculos: Nenhum

Como realizar

  1. Braços estendidos;
  2. Apoiar as mãos em uma parede;
  3. Manter a coluna prolongada;
  4. Empurre os quadris para o lado de forma lenta;
  5. Nessa posição entre 20 a 30 segundos, ou conforme o tempo orientado pelo professor.
