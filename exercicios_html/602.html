Tríceps Testa na Polia Baixa

O exercício trabalha o fortalecimento e hipertrofia dos músculos tríceps, com enfoque o tríceps braquial.

Ficha Técnica

Tipo: Musculação

Grupo Muscular: Tríceps

Aparelho: Nenhum

Músculos: Nenhum

Como realizar

  1. Posicione um banco de exercício em frente a uma polia baixa equipada com um puxador reto. Deite-se no banco com a cabeça próxima ao aparelho;
  2. Segure a barra com as palmas das mãos voltadas para fora;
  3. Estenda os braços totalmente para que eles formem um ângulo de 90 graus com o tronco;
  4. Flexione os cotovelos para baixar o peso enquanto mantém a parte superior do braço fixa;
  5. A barra voltará para a posição de repouso, logo acima da testa;
  6. Faça uma pausa por um momento e, em seguida, flexione os tríceps retornando o peso para a posição inicial.
