Desenvolvimento Arnold

Exercício para fortalecimento e hipertrofia dos músculos da região dos ombros, trabalhando também, músculos auxiliáreis do trapézio. Estimula a coordenação motora.

Ficha Técnica

Tipo: Musculação

Grupo Muscular: Ombro

Aparelho: Nenhum

Músculos: Nenhum

Como realizar

  1. Sente em um banco com encosto para as costas e segure um haltere em cada mão;
  2. Flexione os cotovelos, trazer eles até ficarem próximos ao peito com as palmas das mãos voltadas para o corpo;
  3. Com os halteres na altura do pescoço e, com ajuda dos músculos dos ombros, empurre os pesos para cima e, simultaneamente, gire os punhos para o lado de dentro;
  4. Ao topo do movimento, os braços deverão ficar estendidos;
  5. Retorne à posição inicial de forma controlada e repita os movimentos.
