Torção Deitada

Exercício de alongamento para as lombar e oblíquos, preparando para a atividade física.

Ficha Técnica

Tipo: Mobilidade

Grupo Muscular: Costas

Aparelho: Nenhum

Músculos: Nenhum

Como realizar

  1. Deite com a barriga para cima;
  2. Cruze as pernas de forma que um dos pés esteja apoiado um pouco acima do joelho;
  3. Desloque as pernas para um dos lados mantendo o tronco apoiado no solo.
