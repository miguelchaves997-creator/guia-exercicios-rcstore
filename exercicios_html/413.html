Kipping Bar Muscle-Up

Esse exercício trabalha os músculos do ombros, braços, quadris e núcleo.

Ficha Técnica

Tipo: CrossFit

Grupo Muscular: Corpo

Aparelho: Nenhum

Músculos: Nenhum

Como realizar

  1. Agarre-se na barra com as mãos na largura dos ombros;
  2. Empurre o peito para fora para trazer o corpo para a frente;
  3. Puxe os ombros para trás para inverter a curva do corpo;
  4. Puxe-se para cima depois de balançar 2-3 vezes;
  5. Afaste o queixo pelo menos 2-3 polegadas (5,
  6. 1-7,6 cm) da barra;
  7. Abaixe-se para mover-se para outra puxada para cima.
